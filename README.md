# Rotur Assistant
A versatile Chrome extension for the Rotur ecosystem that allows you to interact with most of Rotur with ease. Anytime, anywhere.

Features:
- Wallet
- Economic Stats (Calculator Included)
- Detailed Lookup
- Wiki
- Claw
- Easy Account Management
- Semi-Universal Account Switcher across select Rotur services

## Build Instructions
Download the repo, go to chrome://extensions, click "Load Unpacked", and then select the folder you got by downloading this repo. I may publish this to the Chrome Web Store eventually.
