originOS

Link: https://origin.mistium.com

Overview

OriginOS is where it all began. OriginOS is a scratch-based web OS, home to a programming language commonly seen and used within the Rotur community: OSL. For most newcomers to the greater Rotur community, originOS is a common way people interact with Rotur (that or using roturBOT). OriginOS allows you to run OSL apps and programs, which can include games.

OSL

This is being phased out in favor of GOSL. The version of OSL used by OriginOS is now called "OSL Legacy", while what was referred to as GOSL is now the new main version of OSL.

OSL is a JS-based language that has relatively simple syntax. It is what every OriginOS program, even the system programs (example: the settings), are built off of. The docs for OSL can be found here.

GOSL

As of May 3, 2025; This is the new main version of OSL

GOSL is a go-based spinoff of OSL that can run outside of originOS, and can therefore be used for backend stuff. The syntax between OSL and GOSL is largely the same, but both OSL and GOSL each have things that the other doesn't have. Some parts of Rotur, such as the OriginChats DMs server, are even written in GOSL. Overall, GOSL is those who want the speed of Go, but at the same time, want JS-style syntax; in other words, it serves as a nice middle-ground for those transitioning from JS to Go.

ICN

While ICN today is attributed to Rotur as a whole, ICN began on originOS, and is still originOS's primary way of rendering vector-esque elements, such as all the badges on peoples' profiles. The docs for ICN can be found here. Various renderers and editors exist for ICN too, but the most common and accessible one as of now is OriginOS's studio app, which has built-in support for real-time ICN editing.
