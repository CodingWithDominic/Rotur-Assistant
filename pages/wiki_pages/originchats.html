OriginChats

Link: https://originchats.mistium.com

Overview

This is where the greater Rotur community resides today. The main origin server can be found by adding the URL wss://chats.mistium.com to your server list. Some clients will even join that server for you. OriginChats is a Rotur-based alternative to Discord. Unlike Discord, OriginChats allows you to host servers directly on your own hardware. OriginChats is also far more lightweight than Discord, using significantly less RAM. Unlike Discord, there is no OriginChats+, premium, or whatever. All features are included in the box right out the gate.

Clients

A handful of clients exist for OriginChats, with the most popular one being the official one linked above. Each client has its own unique set of features. For example, NexPay being a part of Milo's client. As of now, the current list of (known) clients includes:

roturBOT

This is about the OriginChats bot. For the Discord counterpart of this bot, see Discord (roturBOT).

Shortly after the Rotur community migrated to OriginChats, Mistium's first order of business was to port roturBOT over from Discord to OriginChats. Most commands from the Discord version of roturBOT were ported over, either with no changes or minor changes. While roturBOT on Discord was written in python, roturBOT on originChats was written in GOSL. Unlike roturBOT on Discord, roturBOT's commands only work within the OriginChats origin server. You may see roturBOT in other servers, but without the ability to use its commands. The OriginChats version of roturBOT also allows free users to interact with Rotur AI, though with an extremely barebones model compared to what lite and the paid tiers get.
