ICN Editor

View, edit, and manage ICN files here.

Background:

ICN Docs

Since ICN has consistent syntax and a small amount of commands, Rotur Assistant is able to provide its own version of the ICN documentation. Note that any command or argument that is highlighted in yellow are things that only work within Rotur Assistant's implementation of ICN, and nowhere else. You can toggle support for these special commands in the ICN settings tab.

w

Syntax: w (num)
Example command: w 5
Alias: width, weight

Sets the weight (or width) of any subsequent line-based commands that come after this. Defaults to 1 on Rotur Assistant.

c

Syntax: c (#hex)
Example command: c #0000FF
Alias: color, colour

Sets the color of any subsequent shapes drawn. Defaults to black (#000000) on Rotur Assistant.

scale

Syntax: scale (num)
Example command: scale 2

Sets the proportional scale of subsequent shapes drawn. Defaults to 1 on Rotur Assistant.

dot

Syntax: dot x y
Example command: dot 1 1

Creates a circular dot at the specified location.

line

Syntax: line x1 y1 x2 y2
Example command: line -1 -2 3 4

Creates a straightforward line from point A to point B

rect

Syntax: rect x y width height
Example command: rect 0 0 4 3
Alias: rectangle

Creates a filled rectangle. The width and height stretch outward from the center where the specified x and y is.

square

Syntax: square x y width height
Example command: square 0 0 5 5

Same as rect, but the shape is hollow

tri

Syntax: tr1 x1 y1 x2 y2 x3 y3
Example command: tri 3 3 4 4 2 5
Alias: triangle

Creates a filled triangle between the specified points

cutcircle

Syntax: cutcircle x y rad startangle sweepangle filled
Example command: cutcircle 0 0 6 15 60 filled
Alias: arc

Creates an arc (partial circle or full circle). The sweepangle spreads outward in both directions from the startangle.

ellipse

Syntax: ellipse x y width height_multiplier direction filled
Example command: ellipse 0 0 5 2 0 1
Alias: oval

Creates a hollow oval at the specified position.

curve

Syntax: curve x1 y1 x2 y2 gx gy
Example command: curve 0 0 5 5 0 5

Draws a bezier curve between 2 points, with the 3rd point acting as the reference point the curve gravitates towards

cont

Syntax: cont x y
Example command: cont 5 5

Allows some commands to continue from the last point of the previous shape drawn. This works best with the line command.

move

Syntax: move x y
Example command: move 4 6

Moves the canvas's reference point to the specified point

back

Syntax: back
Example command: back

Resets the canvas's reference point back to 0, 0. This is the only ICN command that doesn't have any arguments

image*

Syntax: image (url) scale
Example command: image https://avatars.rotur.dev/Rotur_Assistant 5

*This command is intentionally left out of the official ICN docs, and not all renderers may support this command.

This command draws an image from the specified image URL, with the 2nd argument being the height of the image (in pixels). The image is always centered at -10, -10 on the canvas no matter what, not even being affected by the move command. This command mainly exists so that developers of OriginOS apps can have an easier time making eye-catching icons for their apps.

Exclusive Commands

Note that anything past this point is exclusive to Rotur Assistant

circle

Syntax: circle x y radius
Example command: circle 4 3 5 filled
Alias: cir

Draws a circle at the specified point. This is basically the dot command, but a little more specific.

cutcircle2

Syntax: cutcircle2 x y radius startangle sweepangle filled (ccw? true/false)
Example command: cutcircle2 4 4 5 90 180 0 true
Alias: arc2

Same as the cutcircle command, except you specify where the arc starts and ends, and goes clockwise or counterclockwise depending on what's specified.

ellipse2

Syntax: ellipse2 x y width height direction filled
Example command: ellipse2 1 1 5 10 45 filled
Alias: oval2

Same as the ellipse command, except the height is no longer dependent on the width.
